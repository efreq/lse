# LSE

Super Mario 3D **L**and **S**ave **E**ditor

# Usage
> You need Universal CRT for running, but it should already installed if you using Windows10 or later

```
lse [OPTION]
```

# Build
built with **mingw-w64-ucrt-x86_64-gcc**
